## 💬 Mogelijke Vragen en Antwoorden

### Technische Vragen

**Q: Waarom Laravel en niet Symfony/CodeIgniter?**
A: Laravel biedt rapid development met built-in features, uitstekende documentatie, en grote community. Perfect voor dit project's requirements.

**Q: Hoe heb je security geïmplementeerd?**
A: Session-based auth, CSRF protection, input validation, per-game API tokens, en IP tracking voor audit trails.

**Q: Wat als de applicatie moet schalen?**
A: Database indexing, pagination, caching strategy, en schaalbare Laravel architectuur maken dit mogelijk.

### Design Vragen

**Q: Waarom een dark theme?**
A: Modern appeal, past bij gaming esthetiek, minder vermoeiend voor ogen, en betere leesbaarheid in escape room omgevingen.

**Q: Hoe responsive is het design?**
A: Mobile-first approach met Flexbox, touch-friendly elements, en progressive enhancement voor verschillende devices.

### Project Vragen

**Q: Wat was de grootste challenge?**
A: Real-time score updates zonder page refresh - opgelost met JavaScript polling en performance optimalisatie.

**Q: Wat zou je anders doen?**
A: Meer user testing in development fase, CI/CD pipeline, en component-based CSS voor betere herbruikbaarheid.
