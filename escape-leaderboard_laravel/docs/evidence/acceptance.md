# Acceptance Criteria & Evidence

## Doel

Beschrijf de acceptatiecriteria van de examenopdracht en verzamel bewijsstukken.

## Acceptance criteria (voorbeeld)

-   De leaderboard toont de top 10 scores
-   API accepteert nieuwe scores via POST en valideert input
-   Admin kan scores bewerken en verwijderen via UI
